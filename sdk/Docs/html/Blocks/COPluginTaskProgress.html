Block Definition

COPluginTaskProgress

A block that allows plugins to notify Capture One that the progress of a task, represented by a COPluginTask object, has been updated.

typedef void (^COPluginTaskProgress) (COPluginTask *task, NSUInteger completed, NSUInteger total, NSString *_Nullable, message )

Discussion

This block is passed to plugins as a parameter to runPluginTask:error:progress: and should be invoked when progress updates are waranted.
