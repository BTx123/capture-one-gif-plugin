COProcessScaleMethod

Describes the method used to scale the image.

Definition

typedef NS_ENUM(NSUInteger, COProcessScaleMethod ) { 
	COProcessScaleMethodFixed = 0,
	COProcessScaleMethodWidth = 1,
	COProcessScaleMethodHeight = 2,
	COProcessScaleMethodWidthByHeight = 4,
	COProcessScaleMethodLongEdge = 5,
	COProcessScaleMethodShortEdge = 6,
};

Constants

COProcessScaleMethodFixed

Resample processed image to a fixed percentage. Less than 100& will downsample, any value above will upsample using interpolation to add pixels.

Declared In COProcessSettings.h.

COProcessScaleMethodWidth

Process image to a fixed width. The height will be scaled automatically.

Declared In COProcessSettings.h.

COProcessScaleMethodHeight

Process image to a fixed height. The width will be scaled automatically.

Declared In COProcessSettings.h.

COProcessScaleMethodWidthByHeight

Process image so that one dimension is resized to fit within the dimensions specified, retaining the original aspect ratio, and irrespective of orientation.

Declared In COProcessSettings.h.

COProcessScaleMethodLongEdge

Process image so that the long edge has a fixed size.
