Overview

The recommended way for a plugin to get configuration parameters and task settings is to pass a list of required settings to Capture One. Capture One will then display them at an appropriate location with the Capture One UI look and feel. When users update a plugin setting in Capture One, Capture One will call back into the plugin and inform it about the new value.

COSettingsBase is the most generic representation of a setting and is the base class of all the setting item classes. It contains the basic properties required by Capture One to display a setting: an identifier, the title and an informativeText.
