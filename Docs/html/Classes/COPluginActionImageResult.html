Overview

COPluginActionImageResult represents the result of a round-trip editing task. A round-trip editing task produces images as result and then passes the paths to those images in the images array of the initWithImages: initializer. If the source images were stored in a catalog, Capture One will import the results into the same collection as the source images.

Note: Use COPluginActionImageResult if the result of a plugin task should be imported back into Capture One.
