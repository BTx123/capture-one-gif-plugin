Overview

COPluginActionPublishResult represents the result of a publish task. It returns an array of URLs to Capture One. Capture One will then present the result by showing a notification with an Show button that opens the first URL in the array of URLs in the brower.

  URL

A URLs pointing to the location of the published results. Could be for example a link to a web service.

@property (nonatomic, strong, nullable) NSString *URL

Discussion

Note: The URL will be opened if the user clicks the Show button in the notification that Capture One displays upon successful completion of a publishing task.
