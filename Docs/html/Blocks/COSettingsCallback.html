Block Definition

COSettingsCallback

A block that allows plugins to notify Capture One that an action must be taken in response to a settings update request.

typedef void (^COSettingsCallback) (COSettingsCallbackAction action, id <, NSCopying , NSSecureCoding >, _Nullable payload)

Discussion

This block is passed to plugins as a parameter to didUpdateValue:forSetting:error:callback: and should be invoked when the plugin logic requires Capture One to take a specific action - such as a refresh of the settings UI.
